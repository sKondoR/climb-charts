Найдена проблема в функции findContourPoints:
1. Функция использует бинарный поиск для нахождения радиуса, но алгоритм не адаптируется к разным значениям heightTarget
2. Начальное значение radius = 2.0 может быть неправильным для разных высот
3. Алгоритм не учитывает профиль функции getHeightAt в разных точках пространства
4. Нет ограничения на максимальное расстояние radius для высот выше 10
5. Функция не оптимизирована для разных высот могилы профиля

Рекомендуется использовать бинарный поиск с ограничениями и адаптивной корректировкой для разных значений heightTarget.
